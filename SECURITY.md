# Security Policy

## Supported Versions

Only the latest `main` branch is actively supported with security fixes.

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |
| older   | ❌        |

## Reporting a Vulnerability

If you believe you have found a security vulnerability in TeamCyberOps Recon,
please **do not** open a public GitHub issue.

Instead, report it privately:

- GitHub: open a [Security Advisory](https://github.com/mohidqx/TeamCyberOps-Recon/security/advisories/new)
- Or contact: [@mohidqx on GitHub](https://github.com/mohidqx)

Please include:

1. A clear description of the issue
2. Steps to reproduce
3. Impact assessment
4. Optional: a suggested fix

We will acknowledge reports within **72 hours** and aim to ship a fix or
mitigation within **14 days** for high/critical issues.

## Responsible Use

This project is a **defensive recon platform** intended for authorized
security testing, bug-bounty programs and educational use.

Using it against systems you do not own or do not have explicit written
permission to test is illegal in most jurisdictions. The maintainers
accept no liability for misuse.
